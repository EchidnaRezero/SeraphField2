시각 언어

어두운 흑청/흑연 계열 표면, 낮은 opacity의 시안 라인, 회청색 유틸리티 아이콘, scanline, 부드러운 vignette를 사용합니다. 시안은 낮은 opacity의 HUD 신호로 제한합니다.

로비

  • 이미지 기반 HUD 배경 위에 카테고리 카드를 배치합니다.
  • 카테고리는 THEORY, PAPER, REPO, IMPLEMENT 네 개를 사용합니다.
  • 홈, 프로필, 검색은 상단 우측 유틸리티 아이콘으로 둡니다.
  • 로비에는 검색 입력창을 두지 않습니다.
  • 데스크탑과 모바일은 같은 요소를 쓰고, 데스크탑에서만 카테고리 카드 묶음의 위치를 조정합니다.

위키 본문

  • 상단 카테고리 아이콘 행은 본문 폭 기준으로 중앙 정렬합니다.
  • 상단 검색은 작은 우측 아이콘으로 둡니다.
  • 데스크탑은 우측 문서 로드맵을 유지합니다.
  • 모바일은 문서 메타데이터 아래에 접이식 문서 로드맵을 둡니다.

검색

  • 검색 범위는 scope chip으로 선택합니다.
  • 검색 결과 카드는 모바일과 데스크탑에서 같은 구조를 사용합니다.
  • 데스크탑에서는 결과 칩을 카드 우측에 정렬할 수 있습니다.
